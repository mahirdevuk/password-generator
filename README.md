# Secure Password Generator

## Description
This project is a simple Password Generator website built using HTML, CSS, and JavaScript. It allows the user to generate a random password by clicking a button. The page has a clean design and is easy to use.

## Features
- Generates a random password instantly
- Uses uppercase letters, lowercase letters, numbers, and symbols
- Displays the password in a text box
- Simple and clean user interface
- Styled in a dark theme for a modern look
- Built in one single file for simplicity

## How It Works

### HTML
The HTML creates the structure of the webpage. It includes:
- A title for the page
- A text box to display the generated password
- A button that the user clicks to generate a password

### CSS
The CSS is used to style the webpage. It:
- Sets the background colour and text colours
- Centers the content on the page
- Styles the input box and button
- Makes the design look modern and professional

### JavaScript
The JavaScript adds the functionality. It:
- Stores a group of characters that can be used in the password
- Randomly selects characters from that group
- Creates a password with a set length
- Displays the generated password inside the text box when the button is clicked

## Technologies Used
